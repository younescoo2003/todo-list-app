# 📝 To-Do List App

A simple command-line to-do list application to manage your daily tasks.

## 🚀 Features

- ✅ Add new tasks
- 📋 View all tasks
- ✓ Mark tasks as complete
- 🗑️ Delete tasks
- 💾 Tasks stored in memory

## 🛠️ How to Run

### Prerequisites
- Python 3.x installed

### Steps

1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/todo-list-app.git
cd todo-list-app

python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
